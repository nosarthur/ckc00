# classmate locator

A web application that visualizes the location of CKC00 alumni. 

## Features

* anonymous 
    * display alumni on US map
    * query using alumnus name
    * query with sex and class
* registered user
    * display alumni names
    * view alumni profiles
    * edit profiles
